Node.js examples for the SoSmart API.

Requirements:
- Node.js 18+ for built-in `fetch`

Examples:
- `node examples/admin-login.js`
- `node examples/register-login.js`
- `node examples/trip-create.js`

You can override the default staging API URL with:
- `API_BASE=https://... node examples/admin-login.js`
